# Census Income Project

Project scaffold for training a Census Income classification model and deploying it with FastAPI.

## Structure

- `src/census/`: data processing, model utilities, and slice analysis
- `main.py`: FastAPI entrypoint
- `train_model.py`: training script
- `tests/`: unit and API tests
- `model/`: trained artifacts and slice output
- `.github/workflows/ci.yml`: lint, test, and deployment placeholder
